How It Works

Simplifying Your Event Experience: Our Process Explained

Simple Steps to get started

STEP

01

Create and Configure an Event

Activate your event with the event details, ticket types, and payment info. Then, start promoting it!

STEP

02

Publish the Event

Publishes the event landing page, making it accessible on the internet for users to find and explore.

STEP

03

Ticket Purchase

Visits the event page, adds desired tickets to the cart, and completes the purchase through a standard checkout process.

STEP

04

Order Confirmation

Receives an order confirmation email with a link to download tickets, each containing a QR code for event entry.

STEP

05

QR code scanning

On the event day, presents the QR code, which is scanned to verify and grant access to the event.
